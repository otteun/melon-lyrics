# 🎵 노래 가사 분석 (1990s~2020s)

연대별 한국 노래 가사의 트렌드를 분석하고 시각화한 프로젝트입니다.

---

## 📌 프로젝트 개요

1990년대부터 2020년대까지 연대별 한국 노래 가사를 수집하여, 자주 등장하는 단어를 추출하고 워드클라우드와 빈도 분석으로 시각화합니다. 시대별 가사의 언어적 특징과 트렌드 변화를 탐색하는 것을 목표로 합니다.

---

## 📁 파일 구성

| 파일명 | 설명 |
|---|---|
| `1990s_lyrics.ipynb` | 1990년대 가사 분석 |
| `2000s_lyrics.ipynb` | 2000년대 가사 분석 |
| `2010s_lyrics.ipynb` | 2010년대 가사 분석 |
| `2020s_lyrics.ipynb` | 2020년대 가사 분석 |

---

## 🛠 사용 기술 및 라이브러리

- **Python**
- `matplotlib` — 시각화
- `wordcloud` — 워드클라우드 생성
- **Google Colab** + **Google Drive** — 개발 환경 및 데이터/폰트 관리

---

## 🔍 분석 방법

1. 연대별 가사 텍스트 데이터 전처리
2. 한국어 + 영어 혼합 가사에서 단어 추출
3. 워드클라우드 시각화 (NanumGothic 폰트 적용)
4. 상위 40개 단어 빈도 순위 출력

- 빈도 기준 **상위 40개 단어** 목록

---

## 💡 주요 분석 결과

**전체 트렌드**
- 시대가 흐를수록 노래 가사에서 영어 가사 비중이 높아지는 경향을 확인
- 모든 세대에서 가장 많이 사용된 단어는 **'사랑'**

**연대별 영어 단어 등장 흐름**
- 모든 시대에 공통으로 등장한 단어는 총 **10개**
- 상위 20개 단어 중 영어 단어가 처음 등장한 시점은 **2000년대**
- 영어 단어 **'love'** 는 2010년대부터 2020년대까지 꾸준히 상위 20개에 포함
- 반복되는 영어 단어는 **2010년대**부터 처음 등장
- 2020년대에만 등장한 단어는 **전부 영어**
